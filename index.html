A. First Thing First

1 . Make Repository Locally

Command use to Make pc folder  set as local git hub repository and also allow us to use git  commands: 
git init

2. Set local  Repository user name

This name must not be the same as the remote:
git config --global user.name "your own name"

3. Set local Repository email address

This email address must not be the same as the remote:
git config --global user.name "your own name"

Git Local Repository with Remote Repository

CMD  use to view your repository url:

git remote 

If no ur, it returns nothing

CMD use connet to your Remote:

git remote add origin your_remote_repository_url.git

The origine stand as name alias to  your url

CMD use to push changes that taik place in local repository to be applied or reflect in to the remote repository: 

git push -u origin master

master stand as the remote  destination branch name in our repository.
